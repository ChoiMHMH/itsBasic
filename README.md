## '프로젝트 제목' 소개

- '프로젝트 제목'은 누구나 작성 가능하며, 자유롭게 공유가 가능한 롤링 페이퍼를 제공하는 서비스입니다.
- 누군가에게 보내는 페이지를 작성하거나, 그 페이지에서 보내고 싶은 메시지를 작성할 수 있습니다.
- 말로 못할 감정이 있다면, 이모지를 달아서 표현해보세요!
- 이야기를 전하고 싶은 사람이 있다면, 또 그러한 사람이 많다면 적극 활용해보세요!
